## 📌 Project Overview

Wander & Explore is a modern travel blog website developed using HTML5 and CSS3. The website features a visually appealing hero section, destination showcase cards, an about section, and practical travel tips for travelers. The goal is to provide users with travel inspiration and useful information through a clean and responsive design.

---

## ✨ Features

- Hero section with full-screen background image
- Navigation bar with Destinations, About, Travel Tips, Contact
- Latest Destinations showcase cards
- Practical Travel Tips section
- Clean and responsive design
